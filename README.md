# JD Keyword Extractor – Chrome Extension

JD Keyword Extractor is a Chrome Extension designed to help job seekers instantly identify important keywords from job descriptions and optimize their resumes for Applicant Tracking Systems (ATS).

Applying for jobs often requires tailoring resumes according to each Job Description. This extension simplifies that process by automatically extracting relevant skills and technologies directly from job postings.

---

##  Features

> Extracts important keywords from Job Descriptions  
> Helps improve ATS resume matching  
> Works across company career portals  
> One-click keyword extraction  
> Lightweight and easy to use  

---

##  Problem It Solves

Job descriptions contain numerous technical requirements, making it difficult to identify which keywords should be included in a resume.

Missing critical keywords may result in resumes being filtered out by ATS systems even when candidates possess the required skills.

JD Keyword Extractor bridges this gap by instantly highlighting relevant keywords directly from the job posting page.

---

##  How It Works

1. User opens a job posting page.
2. Extension reads visible job description content.
3. Keywords related to skills, tools, and technologies are detected.
4. Extracted keywords are displayed inside the extension popup.
5. Users can copy keywords and update their resumes accordingly.

All processing happens locally within the browser.
